Esercizio 1: Condizioni con numeri interi

Scrivi un programma che prenda in input due numeri interi, value1 e value2. Dopo aver preso i due numeri, il programma deve verificare le seguenti condizioni:

  • Se la somma dei due interi è maggiore del doppio di value1, stampa: 2 * value2 - value1 + value2
  • Se la somma dei due interi è maggiore del doppio di value2, stampa: 2 * value1 - value1 + value2
  • Se la somma dei due interi è minore di 10, stampa: value1
  • Se entrambi value1 e value2 sono negativi, stampa: "Errore"

Se uno solo dei due numeri è negativo, rendilo positivo e poi verifica le condizioni.

IMPORTANTE: se 2 o più condizioni sono vere contemporaneamente bisogna che vengano stampate ognuna delle stampe delle condizioni verificate tranne nel caso di ERRORE, in quel caso va stampato solo il messaggio di errore

Esempio di soluzione (in C):


int main() {
    if (value1 < 0 && value2 < 0) {
        printf("Errore");
    } else {
        if (value1 < 0) {
            value1 = -value1;
        }
        if (value2 < 0)
            value2 = -value2;
        }
        somma = value1 + value2;

        if (somma > 2 * value1) {
            printf("%d",(2 * value2 - value1 + value2));
        } else if (somma > 2 * value2) {
            printf("%d",(2 * value1 - value1 + value2));
        } else if (somma < 10) {
            printf("%d",value1);
        }
    }
}

In questo esercizio, simuleremo una versione semplificata del gioco del Black Jack. Il programma deve:

  • Generare una variabile totaleAccumulato che rappresenta il numero di punti accumulati dal giocatore, inizialmente impostato a 0.
  • Generare una variabile valoreCarta che rappresenta il valore della prossima carta, con valori casuali tra 1 e 11.

Il gioco segue le seguenti regole:

  • Il giocatore può decidere se girare la carta o fermarsi, fino a 3 volte.
  • Se il giocatore supera il punteggio di 21, perde immediatamente.
  • Se un giocatore raggiunge esattamente 21, chiudendo subito il gioco.
  • Dopo tre turni o quando il giocatore decide di fermarsi, si passa al secondo giocatore.
  • Il giocatore con il punteggio più alto alla fine del gioco vince, a meno che qualcuno non abbia totalizzato esattamente 21.

Esercizio 3: Decifrare una data da un intero

Scrivi un programma che prenda un intero di 8 cifre che rappresenta una data nel formato GGMMAAAA e lo decifri nelle sue componenti:

  • Le prime due cifre rappresentano il giorno
  • Le seconde due cifre rappresentano il mese
  • Le ultime quattro cifre rappresentano l'anno

Ad esempio, l'intero 12111450 rappresenta il giorno 12 novembre 1450.

avvisare l'utente con un messaggio in caso l'intero che rappresenta la data da decifrare non rappresenti una data reale o non è scritto nella forma corretta.

Nota: Si accettano solo anni a 4 cifre.

IMPORTANTE: fare attenzione ai casi limite.
